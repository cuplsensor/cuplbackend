PS User API¶

POST /location¶

create a location

Annotate a capturesample with a location

Status Codes:
  • 200 OK – The created location object
  • 400 Bad Request – invalid input, object invalid
  • 401 Unauthorized – Not authorised. Bad HMAC.
PUT /location¶

update a location

Annotate a capturesample with a location

Status Codes:
  • 200 OK –
  • 400 Bad Request – invalid input, object invalid
  • 401 Unauthorized – Not authorised. Bad HMAC.
  • 404 Not Found – Location not found
  • 405 Method Not Allowed – Validation exception
DELETE /location¶

delete a location

Delete a location

Status Codes:
  • 200 OK –
  • 400 Bad Request – invalid input, object invalid
  • 401 Unauthorized – Not authorised. Bad HMAC.
  • 404 Not Found – Location not found
GET /location¶

get a location

Get one location by ID.

Query Parameters:
 
  • id (integer) – location ID.
Status Codes:
  • 200 OK – One location
  • 404 Not Found – location not found
GET /user¶

get a user

Get a user by oauth_id. The user must correspond with the owner of the bearer token.

Query Parameters:
 
  • oauth_id (string) – Auth0 ID of the user.
Status Codes:
  • 200 OK – A user object
  • 400 Bad Request – bad input parameter
POST /user¶

create a user

Create a new user. The oauth_id must correspond with the owner of the bearer token.

Status Codes:
  • 201 Created – user created
  • 400 Bad Request – invalid input, object invalid
  • 409 Conflict – a user with the same oauth_id already exists
GET /capturesample¶

get a capturesample

Get one capturesample by ID.

Query Parameters:
 
  • id (integer) – capturesample ID.
Status Codes:
  • 200 OK – One capturesample
  • 404 Not Found – capturesample not found
GET /box/{serial}/capturesamples¶

get a list of capturesamples

Get a list of capturesamples in time window defined by start and end timestamps. No duplicates will be returned (samples for the same time that appear in overlapping captures).

Parameters:
  • serial (string) – 8 character box serial.
Query Parameters:
 
  • start_timestamp (string) – Start of the time window.
  • end_timestamp (string) – End of the time window.
Status Codes:
  • 200 OK – A list of capturesamples
  • 400 Bad Request – invalid input, object invalid
